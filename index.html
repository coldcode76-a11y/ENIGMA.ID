How To Play

Pilih metode yang sesuai dengan perangkatmu dan ikuti langkah-langkahnya.

🔌
PowerTunnel
ANDROID
  1. Download PowerTunnel APK:
    powertunnel.id.uptodown.com
  2. Install & buka aplikasinya.
  3. Masukkan host berikut di pengaturan:
    dash.gtps.cloud/android/3141
  4. Aktifkan tunnel, lalu buka Growtopia & connect!
📱
Virtual Host
ANDROID
  1. Download Virtual Host APK:
    apkmirror.com → Virtual Hosts 2.1.0
  2. Install & buka aplikasinya.
  3. Masukkan config host dari link berikut:
    dash.gtps.cloud/how-to-play/3141
  4. Aktifkan virtual host, buka Growtopia & play!
🍎
iOS (iPhone/iPad)
IOS
  1. Buka Safari atau browser di iPhone/iPad kamu.
  2. Ikuti panduan resmi dari link ini:
    dash.gtps.cloud/ios/3141
  3. Ikuti instruksi di halaman tersebut sampai selesai.
  4. Buka Growtopia & langsung connect ke Enigma!
🖥️
Windows
WINDOWS
  1. Buka Notepad sebagai Administrator.
  2. Buka file: C:\Windows\System32\drivers\etc\hosts
  3. Tambahkan baris berikut di paling bawah:
91.134.85.13 growtopia1.com
91.134.85.13 growtopia2.com
91.134.85.13 www.growtopia1.com
91.134.85.13 www.growtopia2.com
🍏
Mac OS
MACOS
  1. Buka Terminal di Mac kamu.
  2. Ketik perintah berikut untuk edit hosts file:
sudo nano /etc/hosts
  1. Tambahkan baris berikut:
91.134.85.13 growtopia1.com
91.134.85.13 growtopia2.com
91.134.85.13 www.growtopia1.com
91.134.85.13 www.growtopia2.com
  1. Save dengan Ctrl+X, lalu buka Growtopia!
